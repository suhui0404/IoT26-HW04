# IoT26 TeamJ HW04- Raspberry Pi Web Server LED Control

## 1. Objective

- Use RPI to create a standalone web server with a Raspberry Pi that can toggle two LEDs

## 2. Environment

- Raspberry Pi 5
- Raspberry Pi OS (64-bit)
- Python 3
- gpiozero library

## 3. Circuit

### Components
- Raspberry Pi
- LED x2
- Breadboard

## 4. Files

- Python source code is included in the repository.
- Result images and GIF files are included in the result folder.

## 5. Issues / Troubleshooting

### 1. Code Error

Cause
- Some errors occurred during the Python code execution process.

Solution
- Modified the code and reran the program to solve the issue.

## 6. Conclusion

Through this project, I learned how to build a standalone web server using Raspberry Pi and control LEDs through a web browser. I also understood the basic operation of an IoT system that controls hardware through a network using the Raspberry Pi IP address.

During the project, issues such as code errors and Wi-Fi connection problems occurred, but solving these issues helped improve my understanding of Raspberry Pi environment setup and GPIO control.

As a result, this project provided experience in combining Raspberry Pi and web server technology to implement a basic IoT system for remote device control.
